# Breeze_Hacked for Windows
将"thewisenerd/Breeze_Hacked"的鼠标指针Clone至此，并修改为Windows的鼠标指针样式
### 1、使用方法

将 Breeze_Hacked_Windows 下载到本地。  
右键点击AutoSetup.inf，选择“安装”
